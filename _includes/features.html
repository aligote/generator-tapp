Features

Please check out our gulpfile.js for up to date information on what we support.

  • Compile and minify Jade
  • Compile, compress and concat Stylus
  • Compile, compress and concat CoffeeScript
  • CSS Autoprefixing
  • Built-in preview server with LiveReload
  • Image optimization
  • Wire-up dependencies installed with Bower
  • Run server either in develop or production mode
